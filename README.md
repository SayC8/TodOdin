# TodOdin
Simple todo manager written in [Odin](https://odin-lang.org/)

## Setup

```bash
git clone https://github.com/SayC8/tododin
cd tododin

# will require password to create symbolic link to /usr/bin/tododin
bash ./install.sh 

# alternatively run 'odin build . -out:tododin'
# if you would prefer to use the binary elsewhere
```
